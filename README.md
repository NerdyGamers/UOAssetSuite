# UO Asset Suite

A comprehensive C# WPF application for managing Ultima Online client files.

## Features
- Art, Landtiles, Animations, Gumps, Maps, Tiledata management

## Setup
1. Download UOFiddler from https://github.com/potential1/UOFiddler
2. Copy Ultima.dll to lib folder
3. Open UOAssetSuite.sln in Visual Studio
4. Build and run
